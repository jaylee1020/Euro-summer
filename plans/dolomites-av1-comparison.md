## AV1-4N vs AV1-4C 비교

| 항목 | AV1-4N (North) | AV1-4C (Central) |
|------|----------------|------------------|
| **구간** | Lago di Braies → Cinque Torri 부근 | Passo Falzarego → Passo Duran 부근 |
| **난이도** | 쉬움~보통 (Easy-Moderate) | 보통~중상 (Moderate-Challenging) |
| **핵심 테마** | 편안함 & 클래식 돌로미테 | WWI 역사 & 드라마틱한 절벽 |
| **지형** | 비교적 완만 | 더 험준, 기술적 구간 있음 |
| **숙소 편안함** | 높음 (개인실 옵션 가능) | 보통 (개인실 제한적) |
| **인파** | 인기 구간, 더 붐빔 | 남쪽으로 갈수록 한적 |
| **하루 하이킹** | 약 4~6시간 | 약 5~8시간 |
| **추천 대상** | 알프스 첫 경험, 편안함 선호 | 경험 있는 하이커, 도전 선호 |

## AV1-4N (North) 상세 일정

| 일차 | 구간 | 숙소 | 거리/시간 |
|------|------|------|-----------|
| Day 1 | Lago di Braies → Rifugio Biella | **Rifugio Biella** (2,300m) | ~6.4km / 3.5h |
| Day 2 | Rifugio Biella → Rifugio Lavarella | **Rifugio Lavarella** (2,060m) | ~13km / 6h |
| Day 3 | Rifugio Lavarella → Rifugio Lagazuoi | **Rifugio Lagazuoi** (2,752m) | ~17km / 6-8h ⚠ 가장 힘든 날 |
| Day 4 | Rifugio Lagazuoi → Cinque Torri / 하산 | Cortina 방면 | ~7-12km / 2-7h |

**하이라이트:**
- Lago di Braies (인스타 유명 호수)
- Fanes 고원 통과
- Rifugio Lagazuoi에서 360도 파노라마 (AV1 최고 고도)
- Cinque Torri (다섯 암탑)

## AV1-4C (Central) 상세 일정

| 일차 | 구간 | 숙소 | 거리/고도 |
|------|------|------|-----------|
| Day 1 | Passo Falzarego → Averau → Cinque Torri → Passo Giau | **Rifugio Città di Fiume** 또는 Staulanza | ~23km / ⬆1,079m |
| Day 2 | Staulanza → Coldai → Tissi | **Rifugio Coldai** 또는 **Tissi** | ~11km / ⬆946m |
| Day 3 | Tissi → Vazzoler → Carestiato → Passo Duran | **Rifugio Vazzoler** | ~16km / ⬆1,022m ⚠ "brutal" |
| Day 4 | Passo Duran → 하산 | 하산 | - |

**하이라이트:**
- Cinque Torri & Passo Giau
- Lago Coldai (AV1 최고 미호 중 하나)
- Civetta 절벽 바로 아래 트레일 (극적)
- 남쪽으로 갈수록 한적해지는 분위기

## 우리 일정과의 호환성

| 항목 | AV1-4N | AV1-4C |
|------|--------|--------|
| 도비아코에서 시작 | ✅ Lago di Braies 근접 (버스 30분) | ❌ Passo Falzarego로 별도 이동 필요 |
| 종료 후 뮌헨 이동 | Cortina → Dobbiaco → Innsbruck → München (~6-8h) | Passo Duran → Belluno → 기차 (더 복잡) |
| 서경 뮌헨 이동 | 4N 종료 지점에서 뮌헨 접근 더 쉬움 | 4C 종료 지점이 남쪽이라 뮌헨까지 더 먼 거리 |
| 본인 솔로 이동 | Cortina/Innsbruck → 스위스 방면 자연스러움 | Belluno → 베네치아 방면으로 우회 필요 |

### 결론

- **AV1-4N이 우리 일정에 더 적합**: 도비아코에서 바로 시작, 종료 후 뮌헨/인스브루크 접근 용이
- **AV1-4C가 더 도전적이고 극적**: 경치는 더 드라마틱하나 종료 지점이 남쪽이라 이후 이동이 복잡
