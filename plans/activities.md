# Activities & Attractions

## 도시별 액티비티

### 베네치아 (7/12 저녁 ~ 7/13 오전, 반나절)

**추천 루트 (07:00~11:30, Piazzale Roma 출발/복귀)**

| 시간 | 장소 | 소요 | 비고 |
|------|------|------|------|
| 07:00 | Piazzale Roma 짐 보관 | - | €6~8/가방, 버스터미널 옆 |
| 07:15 | 아침 식사 (역 근처 Bar) | 15분 | 카푸치노+코르네토 €3~4 |
| 07:30 | 리알토 다리 (Ponte di Rialto) | 도보 20분 | 아침이 한산, 사진 찍기 좋음 |
| 08:00 | 리알토 시장 | 30분 | 월요일 축소 운영 주의 |
| 08:30 | 산마르코 광장 + 대성당 외관 | 도보 20분 | 아침 시간대 가장 한산 |
| 08:40 | 탄식의 다리 | 10분 | 산마르코 바로 옆 |
| 09:00 | 산마르코 대성당 내부 (선택) | 30~45분 | 무료, 09:30 개장. Pala d'Oro €5 |
| 09:45 | 대운하 바포레토 1번 | 40분 | San Marco → Ferrovia. 편도 €9.50 |
| 10:30 | 역 근처 복귀 | - | 치케티 또는 조각 피자로 점심 |
| 11:30 | 짐 찾기 → 역 이동 | - | |

**식사 팁:**
- 치케티 (베네치아식 타파스): 바카로(Bacaro)에서 €1~3/접시, 와인 €2~3
- Pizza al Taglio (조각 피자): €3~5
- (!) San Marco 광장 카페는 커피 €10~15 + 좌석료 → 감상만

**베네치아 → 도비아코 이동 (오후):**
- **추천: 기차** Venezia S.L. → Fortezza 환승 → Dobbiaco
- 소요: 약 4.5~5.5시간 | 가격: €25~55/인
- 주요 출발: 12:30~13:30 → 17:30~18:30 도착
- Trenitalia 앱에서 60일 전부터 예매 (Super Economy 50% 할인)
- FlixBus는 도비아코 직행 드묾, 기차가 더 안정적
